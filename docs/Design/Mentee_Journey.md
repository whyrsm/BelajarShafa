### Mentee Journey: Join Class & Complete Course

1. **Receive invitation**
   - Mentee dapat kode undangan dari mentor (via WA/email/offline)

2. **Join class**
   - Input kode di form "Gabung Kelas"
   - Lihat detail kelas (nama, deskripsi, mentors, jadwal)
   - Klik "Gabung" → konfirmasi sukses

3. **View class dashboard**
   - Lihat upcoming sessions
   - Lihat list mentor
   - Akses forum diskusi

4. **Attend session & check-in**
   - Terima reminder H-1 dan 1 jam sebelum sesi
   - Saat window check-in dibuka, klik tombol "Check-in Sekarang"
   - Konfirmasi: "Anda sudah check-in untuk [Nama Sesi]"

5. **Assigned course**
   - Terima notifikasi: "Course baru ditugaskan: [Nama Course]"
   - Klik notifikasi → langsung ke course detail

6. **Complete course**
   - Browse topics dan materials
   - Tonton video (track progress otomatis)
   - Download/baca dokumen
   - Kerjakan quiz
   - Lihat progress bar naik

7. **Get certificate**
   - Setelah 100% complete, muncul "Selamat! Anda telah menyelesaikan course"
   - Download certificate PDF
